
Функціональний інтерфейс - інтерфейс, який містить
лише один абстрактний метод. Може мати лише одну
функціональність.

Починаючи з Java 8, лямбда-вирази можна використовувати
для представлення екземпляра функціонального інтерфейсу.

Функціональний інтерфейс може мати будь-яку кількість
default та static методів.

Функціональний інтерфейс називають ще
SAM (Single Abstract Method) інтерфейс.
