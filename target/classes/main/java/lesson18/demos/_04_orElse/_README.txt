
orElse() та orElseGet() працюють подібним чином
для повернення альтернативного значення (за замовчуванням),
якщо в об’єкті Optional немає жодного значення.

Але є незначна відмінність.

Метод, наданий в orElse(), завжди виконується незалежно
від того, яке значення зберігається в об’єкті Optional.

З іншого боку, orElseGet() приймає лямбда-вираз типу
Supplier і викликається тоді якщо значення відсутнє в
об’єкті Optional.
Тому orElseGet() працює набагато краще, ніж orElse().
