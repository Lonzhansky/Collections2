
Interface Supplier<T>
https://docs.oracle.com/javase/8/docs/api/java/util/function/Supplier.html

Interface Supplier являє собою операцію,
яка не приймає аргументів і видає один
результат.

Одиночне значення «надається тільки при
необхідності» і без будь-яких переданих
аргументів.

Якщо пишемо лямбда-вираз, якому потрібно
значення, що повертається, то цей лямбда-вираз
можна записати як реалізацію функціонального
інтерфейсу Supplier.

Інтерфейс Supplier надає відкладене
виконання (deferred execution), іноді зване
"ледачою оцінкою" ("lazy evaluation").

Відкладене виконання - відстрочка виконання коду
до тих пір, поки не стане відомо, що виконання
цього коду потрібно.
