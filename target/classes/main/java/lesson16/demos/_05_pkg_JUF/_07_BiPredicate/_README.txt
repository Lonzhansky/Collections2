
Interface BiPredicate<T,U>
https://docs.oracle.com/javase/8/docs/api/java/util/function/BiPredicate.html

Interface BiPredicate є функцією
з логічним значенням, яка приймає два
аргументи і повертає або true, або false.

Якщо пишемо лямбда-вираз, який приймає
два аргументи і використовує ці аргументи для
оцінки умови, що повертає true чи false,
то цей лямбда-вираз можна записати як реалізацію
інтерфейсу BiPredicate
