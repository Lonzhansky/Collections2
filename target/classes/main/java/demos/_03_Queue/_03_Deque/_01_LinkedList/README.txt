
Class LinkedList<E> представляє собою пов'язаний список.
Для внутрішнього представлення даних (елементів) використовує
двонаправлений список.
Class LinkedList є реалізацією двозв’язаного списку для Interface List
та Interface Deque.
E – тип елементів у цьому списку.
Методи, які визначені в Interface Queue та Interface Deque можуть
реалізовуватись Class LinkedList.
https://docs.oracle.com/javase/8/docs/api/java/util/LinkedList.html

Class LinkedList:
- Може містити елементи, що повторюються.
- Дозволяє null.
- Підтримує порядок вставки.
- Несинхронізований.
- Маніпулювання виконується швидко, тому що зсув не потрібен.
- Може використовуватися як список або черга.
