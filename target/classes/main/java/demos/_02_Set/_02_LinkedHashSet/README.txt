
Class LinkedHashSet<E> - хеш-таблиця та реалізація
пов'язаного списку для Interface Set з передбачуваним
порядком ітерацій.
Class LinkedHashSet успадковує Class HashSet, та відрізняється
від Class HashSet тим, що підтримує двозв'язковий список, що
проходить через всі його записи/елементи.
Class LinkedHashSet визначає порядок ітерацій, який є порядком,
у якому елементи були вставлені в набір.
E - тип елементів, що підтримуються цим набором.
https://docs.oracle.com/javase/8/docs/api/java/util/LinkedHashSet.html

При додаванні елементів, Class LinkedHashSet викликатиме equals() і не
вставлятиме елемент, який еквівалентний елементу, який вже є в наборі.

Class LinkedHashSet:
- Містить унікальні елементи.
- Забезпечує операції над множинами/наборами, які є в його ієрархії класів.
- Дозволяє null елементи.
- Несинхронізований.
- Підтримує порядок вставки.
