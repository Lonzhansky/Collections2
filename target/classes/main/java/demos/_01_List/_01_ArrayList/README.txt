
Class ArrayList<E> - реалізація Interface List зі змінним розміром
масиву. Цей клас має певні методи для маніпулювання елементами списку.
E – тип елементів у цьому списку.
https://docs.oracle.com/javase/8/docs/api/java/util/ArrayList.html

Class ArrayList:
- Може містити елементи, що повторюються.
- Дозволяє null.
- Підтримує порядок вставки.
- Несинхронізований.
- Дозволяє довільний доступ, тому що масив працює на основі індексу.
- У класі маніпулювання повільне, оскільки видалення будь-якого
елемента зі списку потребує певну кількість змін.
