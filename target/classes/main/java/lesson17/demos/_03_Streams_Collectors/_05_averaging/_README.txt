
Якщо потрібне середнє значення
елементів Stream, у класі Collectors
є службові методи, що дозволяють це
зробити.

Collectors.averagingInt() - щоб отримати
середнє значення потоку Integer значень.

Collectors.averagingLong() - щоб отримати
середнє значення потоку Long значень.

Collectors.averagingDouble() — щоб отримати
середнє значення потоку Double значень.
