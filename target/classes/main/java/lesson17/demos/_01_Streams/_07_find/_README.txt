
Метод findFirst() Java Stream API повертає
ПЕРШИЙ елемент потоку.
Термінальна операція.
Метод findFirst() повертає Optional, що описує
перший елемент цього потоку, якщо потік не порожній,
повертає порожній Optional, якщо потік порожній.


Метод findAny() Java Stream API повертає
БУДЬ-ЯКИЙ елемент потоку.
Можна вибирати будь-який елемент у потоці.
Термінальна операція.
Метод findAny() повертає Optional, що описує
деякий елемент потоку, якщо потік не порожній,
повертає порожній Optional, якщо потік порожній.
