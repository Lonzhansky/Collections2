
Heap sort (Сортування купою або
Пірамідальне сортування)

https://en.wikipedia.org/wiki/Heapsort

Будуємо з масиву купу, по черзі витягуємо мінімум
купи.

Пірамідальне сортування є методом сортування,
який інтерпретує елементи в масиві, як майже
повне бінарне дерево.

Бінарне/Двійкове дерево
https://uk.wikipedia.org/wiki/%D0%94%D0%B2%D1%96%D0%B9%D0%BA%D0%BE%D0%B2%D0%B5_%D0%B4%D0%B5%D1%80%D0%B5%D0%B2%D0%BE

Алгоритм бере елементи масиву та вставляє їх у
піраміду.
Після побудови піраміди, з неї по черзі віддаляються
найбільші елементи та вставляються в кінець масиву,
де ті знаходяться у відсортованому вигляді.
