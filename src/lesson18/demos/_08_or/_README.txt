
Метод or() введено в Java 9.
Якщо значення присутнє, повертає Optional, що описує значення,
інакше повертає Optional, створений функцією, що надана.

Optional має методи orElse() та orElseGet() для повернення
значення за умовчанням, коли Optional порожній.

Це значення за замовчуванням є розгорнутим значенням замість
Optional.

А якщо нам потрібно повернути Optional за замовчуванням?
Ми повинні самі загорнути це в Optional.

У Java 9 додано метод or(), який дозволяє передавати
постачальника (Supplier), який повертає Optional.
