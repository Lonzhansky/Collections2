
Counting sort (Сортування підрахунком)

https://en.wikipedia.org/wiki/Counting_sort

Алгоритм на основі цілих чисел (тобто передбачається,
що значення вхідного масиву є цілими числами),
без порівняння та лінійного сортування.

Сортування підрахунком створює кошик (bucket) для
кожного значення та зберігає лічильник у кожному кошику.
Потім щоразу, коли у вхідний колекції зустрічається
значення, що відповідний лічильник збільшується.

Оскільки алгоритм сортування підрахунком створює
сегмент для кожного значення, існує накладне обмеження,
яке полягає у тому, що максимальне значення у вхідному
масиві відомо наперед.
Як тільки кожне значення вставлено в кошик, просто
переглядаємо масив лічильника та друкуємо їх залежно
від їх частоти.

Наприклад, якщо вхідний масив містить 0 (нуль) п'ять
разів, то в нульовому індексі масиву лічильника буде 5.
Тепер можна надрукувати нуль 5 разів, перш ніж
надрукувати 1, залежно від його кількості.
Таким чином отримаємо відсортований масив.
