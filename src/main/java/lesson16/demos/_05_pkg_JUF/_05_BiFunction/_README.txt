
Interface BiFunction<T,U,R>
https://docs.oracle.com/javase/8/docs/api/java/util/function/BiFunction.html

Interface BiFunction є операцією,
яка приймає два аргументи та видає один
результат.

Якщо пишемо лямбда-вираз, який приймає
два аргументи одного або різних типів і повертає
значення іншого типу, то цей лямбда-вираз можна
записати як реалізацію інтерфейсу BiFunction.
