
Interface Predicate<T>
https://docs.oracle.com/javase/8/docs/api/java/util/function/Predicate.html

Interface Predicate є функцією
з логічним (boolean) значенням, що повертає
або true або false.

Якщо пишемо лямбда-вираз, який приймає
один аргумент і використовує цей аргумент для
оцінки умови, що повертає true чи false,
то цей лямбда-вираз можна записати як реалізацію
інтерфейсу Predicate.
