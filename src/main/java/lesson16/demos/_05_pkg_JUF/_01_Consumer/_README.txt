
Interface Consumer<T>
https://docs.oracle.com/javase/8/docs/api/java/util/function/Consumer.html

Функціональний інтерфейс Consumer представляє
собою операцію, яка приймає один аргумент
та не повертає результату.

Інтерфейс Consumer надає відкладене
виконання (deferred execution), іноді зване
"ледачою оцінкою" ("lazy evaluation").

Відкладене виконання - відстрочка виконання коду
до тих пір, поки не стане відомо, що виконання
цього коду потрібно.
