
Interface Function<T,R>
https://docs.oracle.com/javase/8/docs/api/java/util/function/Function.html

Interface Function являє собою операцію,
яка приймає один аргумент і видає один
результат.

Якщо пишемо лямбда-вираз, який приймає
один аргумент одного типу та повертає значення
іншого типу (або навіть того ж типу), то цей
лямбда-вираз можна записати як реалізацію
інтерфейсу Function.
