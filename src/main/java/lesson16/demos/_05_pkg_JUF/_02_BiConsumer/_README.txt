
Interface BiConsumer<T,U>
https://docs.oracle.com/javase/8/docs/api/java/util/function/BiConsumer.html

Функціональний інтерфейс BiConsumer представляє
собою операцію, яка приймає два аргументи
та не повертає результату.

Якщо пишемо лямбда-вираз, який вимагає два
аргументи і не повертає значення, то це
лямбда-вираз можна написати як реалізацію
функціонального інтерфейсу BiConsumer.
