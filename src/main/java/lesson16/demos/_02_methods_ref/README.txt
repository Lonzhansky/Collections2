
Посилання на метод (Method reference) - скорочений запис
лямбда-виразу для виклику методу.

Наприклад, якщо лямбда-вираз виглядає так:

str -> System.out.println(str)

Посилання на метод так:

System.out::println

Оператор :: використовується у посиланні на метод для
відділення класу чи об'єкта від імені методу.
