
Class ArrayDeque<E> - клас двонаправленої черги, в якій можемо зробити
вставку та видалення як на початку колекції, так і на її кінці.
Е - тип елементів, що містяться в цій колекції.
Class ArrayDeque є реалізацією Interface Deque із змінним розміром масиву.
У таких масивів немає обмежень щодо ємності, вони ростуть у міру потреби
для підтримки використання.
Недозволено null елементи. Не є потокобезпечним за відсутності зовнішньої
синхронізації.
Class ArrayDeque може також реалізувати Interface Queue.
Методи, які визначені в Interface Queue та Interface Deque можуть
реалізовуватись Class ArrayDeque.
