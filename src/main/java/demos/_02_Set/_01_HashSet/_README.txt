
Class HashSet<E> реалізує Interface Set, що підтримується
хеш-таблицею (фактично екземпляром HashMap).
E - тип елементів, що підтримуються цим набором.
https://docs.oracle.com/javase/8/docs/api/java/util/HashSet.html
https://en.wikipedia.org/wiki/Hash_table

Class HashSet:
- Зберігає елементи за допомогою механізму хешування.
- Містить тільки унікальні елементи.
- Допускає null значення.
- Несинхронізований.
- Не підтримує порядок вставки, елементи вставляються на основі їхнього хеш-коду.
- Добре підходить для пошукових операцій.
