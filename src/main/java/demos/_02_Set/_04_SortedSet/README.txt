
Interface SortedSet<E> - набір, який додатково забезпечує загальний
порядок своїх елементів. Елементи впорядковуються з використанням їх
природнього порядку або за допомогою компаратора, який зазвичай надається
під час створення відсортованого набору.
Ітерація проходитиме набором у порядку зростання елементів.
Interface SortedSet є аналогом SortedMap.
E - тип елементів, що підтримуються цим набором.
Interface SortedSet успадковує Interface Set та може реалізовуватися
через Class TreeSet.
https://docs.oracle.com/javase/8/docs/api/java/util/SortedSet.html


Interface Comparator<T>
https://docs.oracle.com/javase/8/docs/api/java/util/Comparator.html
Функція порівняння, яка накладає загальне впорядкування на деяку колекцію
об’єктів.
Компаратори можна використовувати для керування порядком певних структур
даних (наприклад, SortedSet або SortedMap) або для забезпечення порядку
для колекцій об’єктів, які не мають природного порядку.
