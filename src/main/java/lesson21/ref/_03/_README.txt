
Використовуючи Java Reflection API, можемо отримати інформацію
про конструктори класу, а також створити новий екземпляр
класу.

У Java Reflection клас java.lang.reflect.Constructor
має методи для пошуку конструкторів, отримання модифікаторів
конструктора та створення нового екземпляра.
https://docs.oracle.com/javase/8/docs/api/java/lang/reflect/Constructor.html
