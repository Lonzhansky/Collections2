
Collectors.teeing(), доданий до Java 12.

Метод teeing() дозволяє створити композицію
з двох колекторів, а також є третій аргумент,
поєднувальна функція.

Кожен елемент, переданий в метод, обробляється
збирачами, результати яких потім об'єднуються
за допомогою зазначеної функції злиття в
остаточний результат.
