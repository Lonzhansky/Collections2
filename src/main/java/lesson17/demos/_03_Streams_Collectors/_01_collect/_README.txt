
При використанні Java Streams, в основному,
у нас буде Collection як джерело для потоку,
але можемо зробити протилежне, тобто отримати
Collection з потоку.

Для цього можна використовувати метод collect()
в Java Stream API.

Метод collect() виконує змінну (mutable) операцію
скорочення (reduction) елементів цього потоку,
яка повертає змінний (mutable) контейнер результату.

Метод collect() є термінальною операцією.

Цей змінний контейнер результатів може бути
класом Collection, таким як ArrayList, HashSet
або StringBuilder.
