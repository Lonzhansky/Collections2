
Бінарне дерево (binary tree) – деревоподібна
структура даних, у якій кожен батьківський
вузол може мати не більше двох дочірніх елементів.

Кожен вузол бінарного дерева складається з трьох
елементів:
- елемент даних.
- адреса лівого нащадка.
- адреса правого нащадка.
