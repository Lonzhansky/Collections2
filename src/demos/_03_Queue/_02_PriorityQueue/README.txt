
Class PriorityQueue<E> - необмежена пріоритетна черга
на основі пріоритетної купи.
Е - тип елементів, що містяться в цій колекції.
Елементи пріоритетної черги впорядковуються відповідно
до їх природного порядку або за допомогою компаратора,
наданого під час побудови черги, залежно від того, який
конструктор використовується.
Class PriorityQueue реалізує Interface Queue.
Методи, які визначені в Interface Queue та Interface Deque можуть
реалізовуватись Class PriorityQueue.
