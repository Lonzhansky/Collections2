
Interface NavigableSet<E> - Interface SortedSet, розширений методами
навігації, які повідомляють про найближчі збіги для заданих
цілей пошуку.
E - тип елементів, що підтримуються цим набором.
Interface NavigableSet успадковує Interface SortedSet та може реалізовуватись
через Class TreeSet.
https://docs.oracle.com/javase/8/docs/api/java/util/NavigableSet.html
