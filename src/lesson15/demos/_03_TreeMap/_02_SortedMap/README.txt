
Interface SortedMap<K,V> - Interface Map, який додатково забезпечує
повне впорядкування своїх ключів. Map впорядковується відповідно до
природного порядку його ключів або за допомогою компаратора, який
надається під час створення SortedMap.
K - тип підтримуваних ключів.
V - тип зіставлених/відображених значень.
https://docs.oracle.com/javase/8/docs/api/java/util/SortedMap.html
https://docs.oracle.com/javase/8/docs/api/java/util/Comparator.html
